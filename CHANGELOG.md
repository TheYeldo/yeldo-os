# Changelog

All notable changes to YeldoOS are documented here. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project uses semantic versioning.

## [1.0.0] - 2026-07-18

### Added

- Original responsive browser desktop and localized boot sequence.
- Persistent window manager, dock, shortcuts, status controls, themes, and personalization.
- About, Projects, GitHub, Terminal, Code Explorer, Skills, Timeline, Contact, Settings, Achievements, and Archive applications.
- Fuzzy command palette and global portfolio search.
- Official GitHub REST integration with server-only tokens, Zod validation, caching, timeout, rate-limit, and fallback states.
- Russian default and complete English localization.
- Notifications, local achievements, keyboard shortcuts, reduced motion, simplified navigation, and mobile full-screen applications.
- Localized SEO metadata, generated Open Graph image, structured data, manifest, icons, sitemap, robots, and conservative offline fallback.
- Unit tests for terminal parsing, window transitions, and GitHub validation.
- Setup, customization, contribution, license, and deployment documentation.
